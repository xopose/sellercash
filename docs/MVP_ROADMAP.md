# MVP Roadmap SellerCash

## MVP-1 (реализовано в текущем репозитории)

- Импорт финансовой ленты из таблиц.
- Единая модель денежных событий.
- Прогноз cashflow на горизонт 30-60 дней.
- Сценарный калькулятор (реклама/цена/закупка).
- Индексация документов в Lucene (OpenSearch BM25).
- Rule-based extraction событий:
  - комиссия,
  - срок выплат,
  - логистика,
  - supply disruption.
- Seller context layer:
  - профиль селлера,
  - карточки SKU/категории/страны/маршрута.
- Relevance + translation layer:
  - сопоставление внешнего сигнала seller context,
  - запись параметризованного эффекта в `external_signal_impacts`.
- Webhook для внешней системы:
  - `POST /knowledge/signals/ingest`.
- Объяснение драйверов прогноза.

## MVP-2 (следующий этап)

- Вероятностный прогноз с калиброванными интервалами риска.
- NER + шаблоны для более точного event extraction.
- Дедупликация и версия финансовых правил.
- Причинно-следственные объяснения "что именно поменялось и на сколько".
- Hybrid search (BM25 + vector) по внешним новостям/отчётам.
- Коннекторы к внешним информационным системам, отраслевым лентам и логистическим источникам.
- Усиление seller context слоя (мульти-SKU чувствительность, товарные замены, риск-профиль поставщика).
- Продвинутый relevance engine (скоринг по источнику, NER entity linking, feedback loop).
- Translation layer 2.0: динамический эффект по времени (затухание/эскалация шока).

## Production hardening

- Async ingestion pipeline + очередь задач.
- Миграции БД (Alembic).
- Набор regression tests и контрактные тесты API.
- Наблюдаемость (structured logs, metrics, traces).
- RBAC + audit trail.
